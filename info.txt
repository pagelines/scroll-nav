== Description ==
<h3>Scroll Nav ~easy to use one page navigation with icon menu builder~</h3>
Scroll Nav allows users to build custom one-page navigation menu. Besides common scroll-to controls it additionally offers default blueprint set that is easy to customize or place on various portions of your page. It offers style &amp; layout management with multiple options. Easy to use scroll to top menu add-on, plus it is compatible to use as a Scroll Spy emplacement on template and content area. It will enable you to make one-page scroll navigation menus with no troubles. In addition, built-in icon menu builder will enable you to easily customize Font Awesome iconic fonts, titles and subtitles of your menus.

<p><h3>Usage and Setup Instructions</h3></p>
<ol>
	<li>Install Scroll Nav section through Pagelines store.</li>
	<li>Go to Pagelines "Drag &amp; Drop".</li>
	<li>"Drag &amp; Drop" Scroll Nav to your page template.</li>
	<li>Add sections to your template area.</li>
	<li>Add Content Box or <a href="http://www.pagelines.com/store/sections/scroll-header-pro/" target="_blank">Scroll Header Pro</a> between sections. Create as many clones as you need.</li>
	<li>Create new page with chosen template.</li>
	<li>Configure scroll headings. (visit Demo page &gt; Configuration for more information)</li>
	<li>On Scroll Nav metatab, select menu template and optionally add icon and subtext to each menu item.</li>
	<li>Go to Site Options &gt; Scroll Nav to customize.</li>
	<li>Blend Scroll Nav into your design and you are good to go.</li>
</ol>
